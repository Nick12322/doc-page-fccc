Introduction

Javascript is a language used all across the web in order to script websites. A lot of features you use everyday on a website wouldn't work without javscript

For example, it would be nearly impossible to create an account and sign into it on a website such as facebook, without javascript.

What you should already know

This guide / documentation assumes you have some working knowledge of web design basics through html and css

The javascript fundamentals however are going to be made as simply to understand as possible, for users of all levels.

Javascript and Java

Javascript and Java are both programming languages, but that's about where the similarities end.

Javascript code is NOT Java code, and the two are NOT interchangable.

Hello World

Here, we are going to write out first "Hello World" using javascript.

function greetMe(yourName) { alert("Hello " + yourName); } greetMe("World");

Copy and paste that code into your scratchpad and watch your first javascript program work!

Variables

Variables are used to store values. You can use any letter, symbol, or number as a variable, to store any information. For example:

var x = "y"

If you were to plug into your scratchpad now alert(x) you should get an alert displaying the letter y

Declaring Variables

There are three ways to declare a variable in javascript. First, the var method:

var x = 42

This can be used to declare both local and global variables.

Second is the let method:

let x = 42

This syntax can be used to declare a block scope local variable.

Finally, a variable can be simply assigned a value, like so

x = 42

This always declares a global variable. It generates a strict JavaScript warning. You shouldn't use this variant.

Data Types

There are seven javascript data types

  • Boolean - True and False
  • null - A special keyword denoting a null value. Javascript is case sensitive, so not the same as NULL, Null, or any other variant
  • undefined - A top-level property whose value is undefined
  • Number - 414, 423, 0, 1
  • String - "Hello World!"
  • Symbol - A data type whose instances are unique and immutable
  • Object
